	as opposed to the above commands, edit the file by keep only the lines that containing "111" string

sed '/^u/d' file
removes all the lines that start with character 'u'.

sed '/x$/d' file
deletes all the lines that end with character 'x'.

sed '/^$/d' file
Delete empty lines or blank lines

sed '/^[A-Z]*$/d' file
Delete lines which are in upper case or capital letters

sed '1d' file > newfile
If you dont wish to delete the lines from the original source file you can redirect the output of the sed command to another file.

sed 's/unix/linux/' geekfile.txt
replaces the word “unix” with “linux” in the file.
By default, the sed command replaces the first occurrence of the pattern in each line and it won’t replace the second, third…occurrence in the line.

sed -i 's/original/new/g' file.txt
	-i = in-place (i.e. save back to the original file)
	s = the substitute command
	g = global (i.e. replace all and not just the first occurrence)

sed 's/unix/linux/2' geekfile.txt
replaces the second occurrence of the word “unix” with “linux” in a line.

sed 's/unix/linux/g' geekfile.txt
replace all the occurrences of the string in the line.

sed 's/unix/linux/3g' geekfile.txt
replaces the third, fourth, fifth… “unix” word with “linux” word in a line.

echo "Welcome To The Geek Stuff" | sed 's/\(\b[A-Z]\)/\(\1\)/g'

echo "Welcome to LikeGeeks page" | sed 's/page/website/'

sed '3 s/unix/linux/' geekfile.txt
replaces the string only on the third line.

sed 'nth,$d' filename.txt
To Delete from nth to last line

sed 'x,yd' filename.txt
To Delete line from range x to y

sed '$d' filename.txt
To Delete a last line

sed -e 's/This/That/; s/test/another test/' ./myfile
To run multiple sed commands, you can use the -e option like this:

sed -f mycommands myfile
You can save your sed commands in a file and use them by specifying the file using -f option.

Grep OR Using \|
grep 'pattern1\|pattern2' filename
grep 'Tech\|Sales' employee.txt

Grep OR Using -E
grep -E 'pattern1|pattern2' filename
grep -E 'Tech|Sales' employee.txt

Grep OR Using grep -e
grep -e pattern1 -e pattern2 filename
grep -e Tech -e Sales employee.txt

Grep AND using -E ‘pattern1.*pattern2’
grep -E 'pattern1.*pattern2' filename
grep -E 'pattern1.*pattern2|pattern2.*pattern1' filename
grep -E 'Dev.*Tech' employee.txt

grep -E 'Manager.*Sales|Sales.*Manager' employee.txt
	grep all the lines that contain both “Manager” and “Sales” in it (in any order).
 
Grep AND using Multiple grep command
grep -E 'pattern1' filename | grep -E 'pattern2'
grep Manager employee.txt | grep Sales
	grep all the lines that contain both “Manager” and “Sales” in the same line.
	
Grep NOT using grep -v
grep -v 'pattern1' filename
grep -v Sales employee.txt

egrep 'Manager|Developer' employee.txt | grep -v Sales
	the following will display either Manager or Developer (bot ignore Sales).
